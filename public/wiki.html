DEROULEMENT DU COMBAT

Chaque combat oppose 2 joueurs avec un deck de 20 cartes. Le terrain comporte 3 slots par joueur.

Debut de tour

  • Vous recevez votre energie du tour (+1 par tour, max 10)
  • Vous piochez 1 carte (max 7 en main)
  • Les passifs de debut de tour s'activent (Aura divine, etc.)

Phase d'action (dans n'importe quel ordre)

  • DEPLOYER — Jouez une creature de votre main sur un slot vide. Coute X energie (= cout mana de la carte)
  • ATTAQUER — Cliquez une de vos unites, puis choisissez "Attaquer". Ciblez un ennemi. Gratuit, 1x par carte par tour
  • POUVOIR — Cliquez une de vos unites, puis choisissez "Pouvoir". Ciblez un ennemi. Coute ceil(mana/2) energie, 1x par carte par combat
  • OBJET — Jouez un objet depuis la main. Effet instantane, la carte est defaussee. Coute X energie

Fin de tour

  • Le poison inflige ses degats
  • Les buffs temporaires se reinitialent
  • L'adversaire joue son tour (IA)

Conditions de victoire

  • L'ennemi n'a plus de cartes (terrain + main + deck vides)
  • Apres 20 tours : celui avec le plus de PV totaux gagne
  • Egalite = match nul (pas de recompense)

CALCUL DES DEGATS

Les degats d'une attaque normale sont calcules ainsi :

Degats = max(ATK - DEF, 1) x modificateur_element

Etapes du calcul

  1. ATK effective = ATK de base + buffs (Feroce, Meute, objets)
  2. DEF effective = DEF de base + buffs (Fortification si PV < 30%)
  3. Degats bruts = max(ATK - DEF, 1) — minimum 1 degat
  4. Modificateur element — x1.5 si fort, x0.75 si faible
  5. Marque — +X degats si la cible est marquee
  6. Fragilite Mage — +1 degat si la cible est un Mage

Mecaniques speciales

  • Bouclier — Absorbe les degats avant les PV. Si degats > bouclier, le surplus va aux PV
  • Poison — Inflige X degats au debut du tour ennemi (ignore DEF et bouclier)
  • Contre-attaque — Quand l'unite est attaquee, reflete X degats a l'attaquant
  • Vol de vie — L'unite se soigne d'un % des degats infliges par attaque
  • Execute — Degats multiplies si la cible est en-dessous d'un seuil de PV
DECK STARTER

Le Deck Starter est un deck de test gratuit disponible pour tous les joueurs. Il contient 20 cartes communes faibles (15 creatures + 5 objets).

  • Les cartes du starter n'appartiennent pas au joueur
  • Toutes les cartes coutent 1-2 mana
  • Ideal pour apprendre les mecaniques de combat
  • Accessible depuis le hub Combat > PvP > Deck Starter
